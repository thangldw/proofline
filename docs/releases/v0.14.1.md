# Proofline v0.14.1

Proofline v0.14.1 closes small usability and lifecycle gaps in Personal Second Brain and Learning
Brain while leaving the larger Team Brain/authentication milestone for later.

## Added

- Notes UI shows deterministic tags and resolved or unresolved wiki-links.
- Backlinks show the linking note, exact lines and immutable source-version identity.
- `GET /api/v1/study-cards/{card_id}/reviews` exposes append-only review history within the active
  workspace.

## Fixed

- Changing only a note title no longer creates a content revision or changes ingestion time.
- Saving unchanged note content reports a metadata-only save instead of claiming a new revision.
- The Study queue requests only cards whose due time has arrived; future reviews no longer appear
  as “due now”.

## Run

```bash
python3 -m venv .venv-proofline
.venv-proofline/bin/pip install proofline-0.14.1-py3-none-any.whl
.venv-proofline/bin/proofline serve --port 0 --data-dir .proofline-data
```

Team Brain, authentication/RBAC, Windows verification, portable export for study/proposals and
external pilot qualification remain deferred.
