# Proofline v0.9.0

`v0.9.0` closes the workspace UX gap and makes the watcher decision evidence-based.

The web sidebar now lists API workspaces. Switching workspace clears open evidence state, refreshes
inventory, memories, jobs and overview, and attaches `X-Proofline-Workspace-ID` to subsequent
search, answer, model-run, settings, import and deletion requests.

The versioned watcher receipt at `evals/benchmarks/folder-watcher-1000-v1.json` measures a synthetic
1,000-file local fixture. Three no-op scans measured 404, 403 and 402 ms (median 403 ms), below the
predeclared 1,000 ms threshold. Proofline therefore retains deterministic polling and does not add
a native watcher dependency. The receipt does not qualify Windows, network filesystems or
production scale.
