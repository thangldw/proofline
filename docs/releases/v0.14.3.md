# Proofline v0.14.3

Proofline v0.14.3 starts the real-model comparison milestone with a credential-safe preflight and
reproducible evaluation contract.

## Added

- A versioned comparison manifest requiring at least one local and one remote generation provider.
- Declared exact model revisions, input/output token prices and fixed prompt versions.
- SHA-256 identities for the extraction and grounded-QA datasets.
- Atomic JSON preflight receipts with Proofline/Git revisions and explicit readiness blockers.
- CLI exit `0` only when every configured endpoint is ready; blocked receipts are still written for
  diagnosis and audit.

## Run

Copy `evals/real-model/comparison-v1.example.json`, replace all placeholders, provide the remote key
through the named environment variable, and run:

```bash
proofline eval-real-model-preflight \
  --plan evals/real-model/comparison-v1.json \
  --output evals/real-model/receipts/preflight.json
```

## Qualification boundary

Preflight proves only that a frozen configuration is complete and its endpoints are reachable. This
release does not yet execute paid model comparisons and contains no real-model quality, latency,
cost or external-pilot evidence.
