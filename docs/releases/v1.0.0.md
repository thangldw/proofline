# Proofline v1.0.0

Proofline v1.0.0 closes the first verifiable decision-memory phase. It is an experimental release
for approved, recoverable test data and is not production-qualified.

## Verifiable provenance

- Deterministic Decision Evidence Package v1 in canonical JSON and ZIP formats.
- Content-hashed source-version, chunk, citation, transformation, artifact, review, and root nodes.
- Offline package verification, local artifact explanation, and content-free package comparison.
- Fail-closed validation for hashes, exact spans, relationships, archives, and workspace boundaries.

## Reliability evidence

- Property and fuzz tests for citation bounds, workspace isolation, round trips, source
  immutability, Markdown, and package archives.
- Every-version database migration tests and ingest/export crash-recovery tests.
- Reproducible provenance conformance and qualified synthetic 1K/10K/100K benchmark receipts.
- Verified local backup, restore, portability, deletion cascade, and integrity paths.

## Included application surface

Immutable local ingestion, exact evidence, local Git sources, deterministic retrieval, grounded
answers, governed decisions, notes, study and proposal workflows, provider-optional AI, the bundled
web launcher, and experimental desktop packaging.

## Limitations

This release does not provide package signatures or an authenticity trust model. It has no PDF
source contract, representative-corpus production scale limit, real Windows lifecycle receipt,
signed/notarized installer, real-model quality report, external pilot outcome, security
qualification, or production support commitment.
