# Proofline v0.14.16

Proofline v0.14.16 makes Evidence Studio artifacts portable without separating generated material
from its proof. It remains an experimental pre-alpha release.

## Included

- Every Studio artifact can be downloaded as an evidence package ZIP.
- The package always contains `evidence-manifest.json` with the immutable source ID/version,
  source-content SHA-256, exact offsets/lines, quote text and quote SHA-256.
- Presentations include a real `.pptx`, infographics include a raster PNG, and data tables include
  CSV. Reports and learning artifacts include Markdown; audio overview includes a narration script;
  video overview includes a self-contained storyboard HTML file.
- Export re-resolves every citation against the immutable source content and fails closed on any
  offset, quote or hash mismatch.

## Boundaries

Audio playback still uses the browser speech engine and the package contains its script, not an
MP3. Video remains an interactive/downloadable storyboard, not a rendered MP4. Model-enhanced
media generation and real-model comparison are not part of this release. Native bundles remain
experimental and production qualification remains open.
