# Proofline v0.14.5

Proofline v0.14.5 adds a NotebookLM-inspired Studio while preserving Proofline's stricter
evidence-first contract.

## Added

- A responsive Studio hub for audio overview, presentation, video overview, mind map, report,
  flashcards, quiz, infographic and data table.
- Deterministic local generation from a selected indexed source, with no API key required.
- Saved artifact previews with interactive slides/storyboards, flashcard reveal, quiz answers,
  browser-local narration and exact-evidence inspection.
- Migration 21 for versioned artifacts and citations. Every item records immutable source identity,
  exact offsets/lines, quote and SHA-256 quote hash.

## Provenance and lifecycle

Creating the same artifact kind again for the same source version is idempotent. A newer source
version creates a distinct artifact, so historical citations remain resolvable. Source deletion
preview reports Studio artifact/citation counts and confirmed deletion cascades through both.

## Media boundary

Audio overview uses the browser's installed speech engine. Video overview is an interactive
storyboard, and presentations/infographics are web previews. This release does not claim
downloadable audio/video, `.pptx`, or raster infographic export.

## Validation

- All nine generators are covered by exact-span, quote-hash, idempotency, revision and cascade
  tests.
- The React flow covers generation and evidence opening.
- The production bundle passed browser interaction and visual comparison at 746 × 860 with no
  console warnings or errors.
