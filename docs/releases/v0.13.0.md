# Proofline v0.13.0

Proofline v0.13.0 adds a bounded Learning Brain: deterministic flashcards backed by exact immutable
source evidence and an inspectable review history.

## Added

- Derive one card from each adjacent `Q:` and `A:` pair in an indexed source.
- Exact answer offsets, lines, source/version identity and SHA-256 quote hash.
- Study screen with reveal-first review and `again`, `hard`, `good`, and `easy` ratings.
- Migration 19 for study cards, due/interval state and append-only review events.
- Idempotent re-derivation, workspace isolation and source deletion impact/cascade coverage.

## Source format

```markdown
Q: What makes a Proofline study card trustworthy?
A: Its answer is copied from and linked to an exact immutable source span.
```

Multi-line or model-invented answers are intentionally not inferred in this release. If no valid
pair exists, derivation fails explicitly and leaves no partial card.

## Run

```bash
python3 -m venv .venv-proofline
.venv-proofline/bin/pip install proofline-0.13.0-py3-none-any.whl
.venv-proofline/bin/proofline serve --port 0 --data-dir .proofline-data
```

Open the emitted loopback URL, import or capture the source, then choose **Study**.

## Limitations

Study-card/review portability is not yet included in the portable JSON export. Raw SQLite backup
includes both tables. AI-generated study questions, sync, notifications and team study remain
planned; Windows and production qualification are still open.
