# Proofline v0.14.17

v0.14.17 is an experimental pre-alpha release.

## Added

- Real-Windows local release tooling for wheel, web bundle, frozen sidecar, MSI, NSIS, target
  receipts, checksums, and direct GitHub publication without CI.
- Frozen private-pilot analysis with SHA-256, version, ID, foreign-key, synthetic-row, and citation
  validation before aggregate metrics are calculated.

## Included product surface

Immutable local ingestion, exact evidence, local Git sources, hybrid retrieval, grounded answers,
governed memory, temporal decisions, notes, study, proposals, Evidence Studio downloads, workspace
isolation, portable archive v2, verified backup/restore, OS keyring, bundled launcher, and an
experimental Tauri macOS application.

## Limitations

This release has no real Windows receipt, signed/notarized installer, real-model quality report,
external pilot outcome, security qualification, or production support. Raw pilot data must remain
outside the public repository.
