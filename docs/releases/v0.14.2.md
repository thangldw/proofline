# Proofline v0.14.2

Proofline v0.14.2 makes immutable note history usable without introducing a rich-text editor.

## Added

- Filter the local note inventory by title or `#tag`.
- List all immutable revisions with version number and content length.
- Open any revision in a read-only preview loaded from its exact source/version identity.
- Display a shortened content hash beside the historical preview.

Historical content is never copied into the current editor automatically, so inspecting an old
revision cannot accidentally overwrite the current source.

## Run

```bash
python3 -m venv .venv-proofline
.venv-proofline/bin/pip install proofline-0.14.2-py3-none-any.whl
.venv-proofline/bin/proofline serve --port 0 --data-dir .proofline-data
```
