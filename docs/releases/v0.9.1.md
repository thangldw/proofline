# Proofline v0.9.1

`v0.9.1` is a visual stability release for the local web workspace.

The web app now uses the selected Sky + Mint system across every existing screen: crisp white and
ice-blue surfaces, pale aqua navigation, accessible azure primary actions, mint provenance and
success states, navy typography, and distinct warning and error colors. Semantic CSS variables keep
the palette consistent across search, decisions, sources, settings, ingestion diagnostics, and
model-run lineage without changing product behavior or evidence provenance.

Responsive QA fixed an existing horizontal overflow at 390 px by constraining search tracks and
using a compact navigation grid. Browser checks cover the Search and Settings routes, desktop and
mobile empty states, console output, and the core contrast pairs. The release keeps the established
desktop sidebar and product information architecture; the selected mock is used as color-system
truth rather than as authorization for a layout redesign.

This remains an experimental pre-alpha release. The local API/CLI artifacts and optimized web
archive have the same support boundary documented in the root README.
