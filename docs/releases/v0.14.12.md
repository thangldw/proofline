# Proofline v0.14.12

Proofline v0.14.12 publishes the source repository as a public experimental pre-alpha project after
explicit approval from its owner.

## Public repository

The repository at `https://github.com/thangldw/proofline` is public. Its default branch is `main`,
GitHub Issues are enabled, and the public description and topics identify Proofline as an
evidence-first, local-first engineering decision memory project.

The root license, contribution guide, Code of Conduct, security policy, support policy and
maintainer policy describe how the public experiment is governed. Issue reports must not include
source contents, databases, credentials, prompts or private model payloads.

## Qualification boundary

Public availability does not make this release production-ready. Windows lifecycle evidence,
signed native installers and update rollback, encrypted retention operations, real-model quality,
external-pilot utility and security qualification remain open gates. This release remains a GitHub
pre-release with best-effort support for the documented experiment.

## Validation

- GitHub API verification of public visibility, default branch, Issues, description and topics.
- Full backend/frontend, lint, formatting, web build, egress and deterministic evaluation gates.
- Installed-wheel lifecycle, portability, backup restore/rollback, integrity and macOS Keychain
  receipt.
