# Proofline v0.14.0

Proofline v0.14.0 adds a bounded Third Brain: evidence-backed action proposals that remain human
governed and cannot autonomously modify sources or accepted memory.

## Added

- Generate proposals through the provider-neutral grounded-answer pipeline.
- Persist only grounded output with its model-run lineage.
- Immutable source/version/chunk citations, exact offsets/lines and quote hashes per proposal.
- Candidate review screen with explicit accept/reject and append-only audit events.
- Migration 20 plus workspace isolation, deletion impact and cascade coverage.

## Trust contract

- Missing provider: no proposal is created.
- Insufficient evidence or invalid model citations: no proposal is created.
- Accepted means “reviewed and accepted as a proposal”; it does not execute an action.
- Deleting one cited source deletes the whole dependent proposal rather than weakening evidence.

## Run

```bash
python3 -m venv .venv-proofline
.venv-proofline/bin/pip install proofline-0.14.0-py3-none-any.whl
.venv-proofline/bin/proofline serve --port 0 --data-dir .proofline-data
```

Configure a generation provider under **Settings**, index relevant evidence, then open
**Proposals** and enter the goal or decision to plan.

## Limitations

Proposals are not autonomous agents, tasks or source write-back. Proposal portability is not yet
included in portable JSON; raw SQLite backup includes proposal and citation tables. Team Brain,
authentication/RBAC, Windows verification and production qualification remain open.
