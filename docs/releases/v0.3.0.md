# Proofline v0.3.0

`v0.3.0` adds the first read-only Git repository ingestion slice for local repositories.

## Added

- `POST /api/v1/git-repositories` registers a local Git root, resolves a revision to an immutable
  commit SHA, and indexes tracked Markdown and UTF-8 text files plus commit metadata.
- Search and grounded-answer citations expose source kind, commit SHA, repository-relative path,
  and exact lines for Git evidence.
- Re-scanning the same commit is idempotent; scanning a later commit preserves historical sources
  and citations.
- Repository inventory and deletion endpoints distinguish Git files from commits and remove all
  derived chunks, indexes, embeddings, memories, evidence, jobs, and content-bearing audit data.
- Migration 14 and synthetic Git fixtures cover upgrade, ingestion, search, history, failure, and
  deletion behavior without external services.

## Boundaries

This release reads only already-local repositories. It does not clone over the network, ingest
uncommitted changes, traverse submodules, authenticate to GitHub/GitLab, or write back to Git.
Proofline remains pre-alpha and is not qualified for production data.
