# Proofline v0.12.0

Proofline v0.12.0 adds the first bounded Personal Second Brain slice while preserving the product's
evidence-first contract. Plain Markdown notes become searchable sources with immutable revisions,
deterministic tags/wiki-links and exact-span backlinks.

## Added

- Quick capture and revision in the local Notes workspace.
- Stable generated `note://` source identity per note.
- Immutable source versions for every content revision; historical content remains resolvable.
- Unicode-aware `#tags` and `[[wiki links]]` with exact offsets and line numbers.
- Backlinks that identify the current linking source version and exact link span.
- Workspace-scoped note API and existing search, backup, portability and deletion behavior.

## Run

```bash
python3 -m venv .venv-proofline
.venv-proofline/bin/pip install proofline-0.12.0-py3-none-any.whl
.venv-proofline/bin/proofline serve --port 0 --data-dir .proofline-data
```

Open the emitted loopback URL and choose **Notes**. Use plain Markdown, `#tags`, and
`[[Wiki links]]`; saving changed content creates another immutable revision.

## Limitations

This is not a rich-text editor, graph canvas or sync service. Links resolve by the current note
title within one workspace, ambiguous duplicate titles remain unresolved, and backlinks describe
current note versions. Learning Brain, grounded
AI planning and Team Brain remain planned in ADR 0004. Windows and production qualification are
still open.
