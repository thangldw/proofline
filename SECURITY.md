# Security Policy

Proofline is pre-alpha and has not received a security audit. Do not expose the
development server directly to the public internet or use it for irreplaceable
or highly sensitive production data.

Please do not file public issues for suspected vulnerabilities. Until a
dedicated security mailbox is published, contact the repository owner through
their private GitHub contact channel. Include affected versions, impact,
reproduction steps, and a minimal proof of concept. Do not include third-party
secrets or personal data.
